### 공통 스펙

**개발 방식**
- UI 디자인: Stitch

- CLAUDE.md + skill/sub-agent/script 최대 활용
- 병렬처리로 토큰 최소화, 결과물 극대화

**공통 기술 스택**

- 프론트: Next.js
- 배포: Vercel

### 1. 웹취약점/테스트 툴

> 보안 + 기능 + 성능을 AI가 병렬로 검증하는 툴

**웹 취약점 탐지**
- OWASP Top 10 기반 자동 스캔
- STRIDE 위협 모델 분석
- SQL Injection, XSS, CSRF, SSRF 등 항목별 상세 리포트
- 취약점 심각도 분류 (Critical/High/Medium/Low)
- 취약점별 해결 코드 예시 제시
- 재스캔으로 수정 여부 자동 검증
- 스캔 히스토리 및 추이 대시보드

**기능 테스트**
- 주요 사용자 플로우 시나리오 자동 생성 및 실행
- 실제 브라우저 기반 E2E 테스트 (Playwright)
- 테스트 결과 스크린샷 자동 캡처
- 실패 원인 AI 분석 및 해결방안 제시
- 테스트 커버리지 리포트

**단위 테스트**
- 소스코드 분석 후 테스트 케이스 자동 생성
- 기존 테스트 코드 품질 분석 및 개선 제안
- 커버리지 목표치 설정 및 미달 알림

**성능 테스트**
- 페이지별 로딩 속도 측정
- Core Web Vitals 측정 (LCP, FID, CLS)
- 부하 테스트 (동시 접속자 시뮬레이션)
- 병목 구간 AI 분석

**리포트 및 알림**
- 종합 리포트 PDF 출력
- Before/After 비교
- 슬랙/Teams/이메일 알림 연동
- sub-agent 병렬 실행으로 취약점/기능/단위/성능 동시 진행

### 2. CI/CD 툴

> AI가 리뷰 · 배포 · 분석까지 자동화하는 배포 관리 툴

**형상관리 연동**
- Git/SVN 연동, 소스 관리 및 배포
- PR, merge, commit, reject 기능
- 환경별 diff 뷰 (dev/staging/prod)

**배포 관리**
- 배포 서버 및 형상관리 서버 간편 세팅
- 클릭만으로 배포/서버 재시작/시작/종료/상태 확인
- 원클릭 롤백 + 롤백 이유 자동 기록
- 배포 히스토리 타임라인

**AI 기능**
- PR 올라오면 AI 코드리뷰 자동 코멘트
- 배포 실패시 로그 AI 분석 및 원인/해결방안 제시
- AI 릴리즈노트 자동생성 (사용자용/개발자용 두 버전)

**알림 및 관리**
- Discord 알림 연동 (배포/실패/롤백 이벤트)
- Discord Webhook URL 등록으로 간편 연동
- 채널별 알림 분리 (배포 채널, 에러 채널 등)
- 프로젝트별 권한관리 (admin/dev/viewer)
- 감사 로그

### 3. 문서 관리 툴

> Notion + Confluence + Swagger를 하나로 합친 전사 AI 지식 허브

**에디터**
- 리치텍스트 + 마크다운 + 블록방식
- 자동저장, 단축어
- 이미지/동영상/파일 첨부 및 임베드
- 테이블, 체크리스트, 토글 블록

**구조 및 관리**
- 프로젝트별 → 폴더/파일 구성
- 태그 시스템, 버전관리, 공개여부
- 문서 만료일 설정
- 템플릿 기능 (회의록, 온보딩 등)
- 즐겨찾기, 최근 본 문서

**검색**
- 전문검색, 태그검색
- AI 기반 의미검색

**소스코드 영역**
- 언어별 신택스 하이라이팅
- 코드 복사, 라인 넘버링
- 코드 diff 뷰
- 인라인 코드 실행 (JS/Python 등)

**API 문서 + 테스트 통합**
- Method/URL/Header/Body 입력
- 환경별 Base URL 설정 (dev/staging/prod)
- 인증 토큰 저장 및 자동 주입
- 요청/응답 히스토리 저장
- 응답 JSON 포맷팅 및 복사
- 문서와 테스트가 같은 페이지

**AI 기능**
- 문서 요약 자동생성
- 소스코드 기반 API 문서 자동생성
- 문서 리뷰 (누락/오류/개선점)
- 코드 변경시 관련 문서 업데이트 알림
- 사내 문서 기반 AI 챗봇
- 학습자료 기반 퀴즈 자동생성
- 다국어 자동번역
- 문서 커버리지 대시보드

**협업**
- 프로젝트별 권한관리 (admin/dev/viewer)
- 댓글, 감사 로그
- 모바일 지원

### 4. FCM 푸시 모듈

> 마이크로서비스로 상시 구동, API 호출로 사용하는 통합 푸시 발송 서비스

**서비스 형태**
- Spring Boot 또는 FastAPI 단독 서버 구동
- 다른 프로젝트에서 REST API 호출로 푸시 발송
- 프로젝트별 API 키 인증

**핵심 API**
```
POST /api/push/send          # 단건 발송
POST /api/push/send/bulk     # 다건 발송
POST /api/push/send/topic    # 토픽 발송
POST /api/push/schedule      # 예약 발송
DELETE /api/push/schedule    # 예약 취소
GET  /api/push/history       # 발송 히스토리
GET  /api/push/stats         # 통계
```

**발송 기능**
- 단건/다건/토픽/그룹 발송
- 예약 발송, 반복 발송, 발송 취소
- 디바이스 토큰 관리 (등록/삭제/갱신)
- 메시지 템플릿 저장 및 재사용
- A/B 테스트
- 다국어 메시지 지원

**AI 기능**
- 발송 내용 입력시 AI가 제목/본문 자동 작성
- 최적 발송 시간대 추천 (오픈율 기반)
- 수신 거부 패턴 분석 및 개선 제안

**관리 UI (심플)**
- FCM 프로젝트 등록 및 API 키 관리
- 발송 히스토리 및 통계 대시보드
- 템플릿 관리
- Webhook 연동 설정

### 5. 이미지 썸네일 생성 툴

> 마이크로서비스로 상시 구동, URL 요청으로 즉시 이미지 변환하는 서비스

**서비스 형태**
- Spring Boot 또는 FastAPI 단독 서버 구동
- URL 파라미터 방식으로 즉시 이미지 리턴
- 결과물 CDN 캐싱으로 중복 요청 최적화

**핵심 API**
```
# URL 파라미터 방식
GET  /thumbnail?url=https://...&w=300&h=200&fmt=webp&q=80

# 업로드 방식
POST /thumbnail/upload

# 다중 사이즈 한번에
POST /thumbnail/multi
```

**변환 기능**
- 목표 사이즈 지정 (width/height, 비율 유지 옵션)
- 확장자 변환 (jpg/png/webp/avif/gif)
- 품질 조절, 배경색 지정
- 크롭 방식 선택 (center/top/bottom/smart)
- 스마트 크롭 (AI 주요 피사체 자동 크롭)
- 워터마크 삽입 (텍스트/이미지, 위치/투명도)
- 이미지 회전/반전
- 필터 적용 (흑백/블러/샤프닝/밝기/대비)
- 다중 사이즈 동시 생성

**성능**
- CDN 캐싱, 중복 변환 방지
- 비동기 처리 + Webhook 콜백 (대용량)
- 변환 큐 및 우선순위 관리
- S3/GCS/Cloudinary 연동

**관리 UI (심플)**
- API 키 발급 및 관리
- 사용량 대시보드
- 변환 히스토리
